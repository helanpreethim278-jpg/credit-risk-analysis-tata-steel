# 📊 Credit Risk Analysis — Tata Steel Limited

## Project Overview
Built a full Altman Z-Score credit risk model for Tata Steel Limited 
using publicly available financial data from Screener.in and NSE India.

## Tools Used
- Microsoft Excel (Advanced Formulas, Conditional Formatting)
- Excel Labs (LABS.GENERATIVEAI for AI Commentary)
- Altman Z-Score Model (1968)

## Model Components
| Sheet | Description |
|---|---|
| INPUTS | Raw financial data entry |
| RATIOS | 5 Altman Z-Score component ratios |
| ZSCORE | Final Z-Score output and credit rating |
| STRESS_TEST | 3-scenario stress testing |
| PEER_COMP | Sector comparison (5 steel companies) |
| DASHBOARD | Executive summary dashboard |

## Key Findings
- **Z-Score: 4.28** — Safe Zone
- **Implied Rating: AAA/AA**
- **Recommendation: BUY / HOLD**
- Under severe recession, Z-Score drops to 1.83 (Grey Zone)

## Data Source
Screener.in | NSE India | FY2026
